1. Real time censoring

WORKING : Working is simple as the censor work is done by three constraints

Personal Virtual Private Network

As you may know VPN works by establishing a digital connection between your computer and a remote server owned by a VPN provider, Here we are not relaying on any VPN provider as we are creating a remote server on our own by cloud computing services.

WORKING : For starters we are cretaing an Virtual machine on the Cloud server, it is called E2 Instance in AWS. Through we create a key pair for SSH access

Now for the big part we are using the Opensource OpenVPN which creates a secure tunnel for data traffic to pass between the VPN client and server. Which makes its accessible so easy.
